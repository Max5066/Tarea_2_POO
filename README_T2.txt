
# Simulador de Patrón Publisher-Subscriber

## Información del Proyecto

### Etapas Desarrolladas
1. **Etapa 1**: Implementación básica del patrón con publicadores y suscriptores de video
2. **Etapa 2**: Integración de reproductor de video en los suscriptores
3. **Etapa 3**: Implementación de publicador GPS con interpolación de posiciones
4. **Etapa 4**: Sistema completo con interfaz gráfica integrada y documentación final

### Repositorio Git
```
https://github.com/[tu-usuario]/publisher-subscriber-simulator.git
```

## Requisitos 
- JDK 21+
- JavaFX 21+

## Instalación y Configuración
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/Max5066/Tarea_2_POO.git
   ```
2. Importar como proyecto en IntelliJ
3. Configurar VM Options para ejecución:
   ```
   --module-path /path/to/javafx-sdk/lib --add-modules javafx.controls,javafx.fxml,javafx.media
   ```

## Generación de Documentación
Para generar la documentación Javadoc:
```bash
mvn javadoc:javadoc
```
