# Simulador de Patrón Publisher-Subscriber

## Requisitos
- JDK 21+
- JavaFX 21+

## Instalación
1. Clonar el repositorio
2. Importar como proyecto en IntelliJ
3. Configurar VM Options:

--module-path /path/to/javafx-sdk/lib --add-modules javafx.controls,javafx.fxml,javafx.media


## Uso
Ejecutar la clase `PubSubsPatternSimulator` como aplicación Java.

### Características implementadas:
- Publicadores de video y GPS
- Suscriptores para cada tipo de contenido
- Interfaz gráfica con JavaFX
- Tiempo de simulación desde cero
